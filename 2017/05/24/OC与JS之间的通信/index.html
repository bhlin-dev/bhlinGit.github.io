OC与JS之间的通信

发表于 2017-05-24 | 分类于 iOS |

####1. oc与js之间的通信可以通过webView的代理进行实现。

-(BOOL)webView:shouldStartLoadWithRequest:navigationType:
在这个代理中webView发送请求之前，都会先调用这个方法（能拦截所有的请求），所以我们可以在js中设置一定的规则在这个代理中进行解析。

  • js调用oc方法：

在OC中的代码
-(BOOL)webView:(UIWebView*)webView shouldStartLoadWithRequest:(NSURLRequest*)request navigationType:(UIWebViewNavigationType)navigationType{ NSString *url = request.URL.absoluteString; NSRange range = [urlrangeOfString:@"bh://"]; NSUInteger loc = range.location; if(loc !=NSNotFound) { NSString*method = [urlsubstringFromIndex:loc + range.length]; SEL sel =NSSelectorFromString(method); [selfperformSelector:selwithObject:nil]; } return YES; }

在JS中的实现以下代码
function callNativeOne() { window.location.href ='bh://callNativeOne'; }

当网页中的按钮点击调用callNativeOne这个js方法时，会在代理中进行拦截，然后去进行判断是否url scheme等于bh，如果等于说明js调用oc中的方法，url scheme后面是方法名。

####2. js调用oc方法，通过iFrame来让js与oc进行交互

在oc中实现以下代理方法

-(BOOL)webView:(UIWebView*)webView shouldStartLoadWithRequest:(NSURLRequest*)request navigationType:(UIWebViewNavigationType)navigationType { NSURL *url = [request URL]; if([[url scheme] isEqualToString:@"bh"]) {//注意url scheme全部被转化成小写 NSArray *params = [url.query componentsSeparatedByString:@"&"]; NSMutableDictionary *tempDic = [NSMutableDictionary dictionary]; for(NSString *paramStrinparams) { NSArray *dicArr = [paramStr componentsSeparatedByString:@"="]; if(dicArr.count >1) { NSString *decodeValue = [dicArr[1] stringByReplacingPercentEscapesUsingEncoding:NSUTF8StringEncoding]; [tempDic setObject:decodeValue forKey:dicArr[0]]; } } returnNO; } returnYES; }

在js中实现以下代码
functioncallNativeTwo() { var url ="bh://test?title=name&pass=value"; var iFrame; iFrame = document.createElement("iframe"); iFrame.setAttribute("src",url); iFrame.setAttribute("style","display:none;"); iFrame.setAttribute("height","100px"); iFrame.setAttribute("width","100px"); iFrame.setAttribute("frameborder","100px"); document.body.appendChild(iFrame); //发起请求后这个iFrame就没有用了，所以把它从dom上移除掉 iFrame.parentNode.removeChild(iFrame); iFrame =null; }

####3. js调用oc方法，通过ios7之后系统新增的JavaScriptCore框架

首先先引入
#import <JavaScriptCore/JavaScriptCore.h>

然后在这个加载网页结束的代理中实现以下代码
-(void)webViewDidFinishLoad:(UIWebView*)webView

JSContext *context = [self.webViewvalueForKeyPath:@"documentView.webView.mainFrame.javaScriptContext"];//获取js的上下文 context[@"callNatives"] = ^() { NSArray*args = [JSContextcurrentArguments]; for(JSValue*jsvalueinargs) { NSLog(@"%@",jsvalue.toString); } };
在js中通过以下方式来调用上面的callNative方法

function callNative() { callNatives('测试a','测试b','测试c'); }

OC调用JS方法：
1.通过webView的stringByEvaluatingJavaScriptFromString:来进行调用js方法

比如：
[self.webView stringByEvaluatingJavaScriptFromString:@"alert('测试数据')"];

stringByEvaluatingJavaScriptFromString该方法返回调用js脚本的结果或者0

2.还是通过JSContext来调用js方法

比如：
JSContext*context = [self.webViewvalueForKeyPath:@"documentView.webView.mainFrame.javaScriptContext"];//获取js的上下文 NSString*textJS =@"alert('这里是JS中alert弹出的message')"; JSValue*value = [contextevaluateScript:textJS];//存在返回结果

通过js调用oc返回值我暂时没有找到，但是可以通过stringByEvaluatingJavaScriptFromString方法来回调到某个js方法并且把返回值回调回去
